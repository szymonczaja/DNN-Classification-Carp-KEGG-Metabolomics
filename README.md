KEGG Data Preprocessing and Dimensionality ReductionThe initial dataset had a significantly higher number of features than samples ($p \gg N$), which required aggressive dimensionality reduction.Correlation Feature Selection: Features that were highly correlated (Pearson correlation $> 0.9$) were handled by removing the one with the lower variance, as it was deemed less informative.Principal Component Analysis (PCA): The final reduction was performed using PCA to condense the data into an optimal number of components while preserving maximum variance.The data was then scaled and normalized for optimal neural network training.2. Deep Neural Network (DNN) ArchitectureA multi-layer DNN model was built using Keras/TensorFlow.ReLU activation was used in the hidden layers, and Softmax activation was applied in the output layer to handle the five classification classes.Hyperparameters (number of layers, neurons, optimizer) were carefully tuned to maximize predictive accuracy.3. Model Evaluation and InterpretationEvaluation Metrics: Model performance was assessed using Accuracy, Loss, and a detailed Confusion Matrix for the multi-class prediction.Model Explainability (PFI): Instead of SHAP, Permutation Feature Importance (PFI) was utilized to interpret the model. PFI helped determine which KEGG metabolic pathways were the most influential predictors, providing critical biological insight into the observed differences caused by supplementation and environment.
